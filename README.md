# E-Commerce Web Application

## Project Description

This project is a simple E-Commerce Web Application that allows users to browse products, add products to the cart, and place orders.

## Features

- Product Catalog
- Add To Cart
- Checkout Functionality
- Backend API using Express.js
- Role-based Structure (Admin/User)
- Database Ready Architecture

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB (Optional)

## How To Run

1. Clone the repository

2. Install dependencies

npm install

3. Start backend

node server.js

4. Open index.html in browser

## Output

- Product listing displayed
- Products can be added to cart
- Total price calculated
- Checkout functionality available

## Learning Outcome

Learned full-stack web development concepts including frontend development, backend APIs, and e-commerce workflow.
